# Task Delegation Prompts

**Domain:** Leadership
**Audience:** AI agents, junior developers, senior specialists

## When to Use

Use these prompts when you need to hand off a task cleanly — to a human or an AI agent — with enough context that they can execute autonomously without coming back with basic questions.

## Prompts in This Category

| File | Situation |
|------|-----------|
| `delegate-to-ai-agent.md` | Writing a task brief for an autonomous AI agent |
| `delegate-to-junior-dev.md` | Assigning a scoped task to a junior developer |
| `delegate-to-senior-specialist.md` | Handing off a complex problem to a domain expert |
| `coordinate-multi-agent-task.md` | Decomposing a task across multiple AI agents in parallel |

## Tips

- Great delegation specifies: goal, constraints, definition of done, and who to contact if blocked.
- For AI agents, also specify: available tools, output format, and what NOT to do.
- Don't over-specify process — specify outcomes and let the delegate own the how.
