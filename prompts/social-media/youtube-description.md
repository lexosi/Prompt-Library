# YouTube Video Description

**Category:** social-media
**When to use:** Writing a YouTube video description that ranks in search, gets clicked, and converts viewers into subscribers.

---

Write a YouTube video description for a developer or game development video. A YouTube description must serve three audiences simultaneously: the search algorithm (keywords), the potential viewer scanning the results page (first 2 sentences), and the existing viewer who clicked and wants more context.

Structure:
1. **First 2 sentences (above the fold):** These appear in search results. Must contain: the primary search keyword, what the video actually shows, and a reason to watch. No fluff.
2. **Video summary (2–3 sentences):** What the viewer will see and learn. Specific outcomes, not vague claims.
3. **Timestamps:** List 5–8 key moments with time codes. Timestamps improve retention and search indexing.
4. **Resources mentioned:** Links to code, tools, documentation, or related content mentioned in the video.
5. **Connect section:** Where to find you — 2–3 links max. LinkedIn, GitHub, Discord. Don't list everything.
6. **SEO keywords (subtle):** Work 3–5 relevant search terms naturally into the description — don't keyword stuff.
