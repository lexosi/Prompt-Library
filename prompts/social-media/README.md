# Social Media Prompts

**Domain:** Content Creation
**Platforms:** LinkedIn, Twitter/X, TikTok, YouTube

## When to Use

Use these prompts when turning technical work, insights, or experiences into platform-native content. Each platform has a distinct voice, format, and audience expectation — these prompts account for that.

## Prompts in This Category

| File | Situation |
|------|-----------|
| `linkedin-technical-post.md` | Sharing a technical insight or lesson on LinkedIn |
| `twitter-thread-ai.md` | Writing a Twitter/X thread on an AI engineering topic |
| `tiktok-dev-content.md` | Scripting a short-form TikTok video about dev or game work |
| `youtube-description.md` | Writing a YouTube video description optimized for search and click-through |

## Tips

- LinkedIn rewards specificity and vulnerability — share real numbers and real failures.
- Twitter/X threads work best when they make one counterintuitive claim in the first tweet.
- TikTok dev content performs best when it has visible code, a clear "wow" moment, and runs under 45 seconds.
- YouTube descriptions should include the key phrases a viewer would search for, in the first two sentences.
