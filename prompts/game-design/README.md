# Game Design Prompts

**Domain:** Game Development
**Platforms:** UEFN (Unreal Editor for Fortnite), Fortnite Creative, Roblox

## When to Use

Use these prompts when designing game mechanics, economies, player experiences, or island concepts for Fortnite or Roblox. They assume familiarity with both platforms' constraints and player demographics.

## Prompts in This Category

| File | Situation |
|------|-----------|
| `fortnite-mechanic-design.md` | Designing a core gameplay mechanic for a Fortnite Creative island |
| `roblox-experience-design.md` | Designing a Roblox experience with clear loop and monetization |
| `uefn-island-concept.md` | Developing a full island concept for UEFN from a one-line idea |
| `game-economy-design.md` | Designing in-game economies for Fortnite or Roblox |

## Tips

- Fortnite Creative / UEFN islands must hook players in the first 30 seconds — design for retention from second one.
- Roblox monetization is driven by Robux purchases — design economies that feel fair while creating clear value in premium items.
- Reference successful published islands or experiences when describing the tone you're aiming for.
