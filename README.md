# Prompt Library — AI Engineering Lead Edition

A bilingual (English + Spanish) collection of production-ready prompts for an AI Engineering Lead working across technical, leadership, business, game design, and content creation domains.

## Who This Is For

This library is built for someone who:

- Leads AI agent systems and human engineering teams
- Works daily with **Rust**, **Python**, and **TypeScript**
- Builds games in **UEFN (Unreal Editor for Fortnite)** and **Roblox**
- Communicates with clients, investors, and technical stakeholders
- Creates technical content for developers and general audiences

If that sounds like you, every prompt here is immediately usable — no placeholders to fill in, no generic "insert your context here" scaffolding.

## Structure

```
prompts/
├── debugging/            Code and agent bug diagnosis
├── code-review/          Code quality and safety reviews
├── architecture-design/  System and AI pipeline design
├── refactor/             Code restructuring and cleanup
├── task-delegation/      Delegating to AI agents and developers
├── project-kickoff/      Starting projects with full alignment
├── decision-making/      Structured technical decision frameworks
├── client-communication/ Client-facing written communication
├── pitch-and-positioning/ Pitching products and ideas
├── outreach/             Cold outreach and partnerships
├── prompt-debugging/     Diagnosing and improving prompts
├── context-preparation/  System prompts and agent context
├── game-design/          Fortnite, UEFN, and Roblox design
├── uefn-scripting/       Verse scripting for UEFN
├── content-strategy/     Thought leadership and courses
└── social-media/         LinkedIn, Twitter/X, TikTok, YouTube
```

## How to Use

1. Browse [`index.md`](index.md) to find the prompt you need.
2. Open the prompt file — each has an English and Spanish version.
3. Copy the version you need, paste it to your AI tool, and go.
